
SM2算法作为ECC算法的国内自产算法，计算过程大体跟ECC并无差距，只是使用特殊参数。本项目对ECC算法中计算标量乘法环节的硬件加速进行了实现。
本设计包含了 SM2 加密中的点乘运算模块以及坐标转换的模除模块。 设计中椭圆曲线的有限域运算使用了乘法器与基于 SCA-256 素数性质设计的模约减器来计算模乘法，速度快于传统的模乘法； 点加和点倍运算通过优化运算顺序，使模加减法尽量与模乘法同周期运算来减少运算时间， 并且使用了双模块并行运算进行了加速。
本设计基于 Xilinx FPGA 平台完成了综合与验证，实验结果表明：所设计的加速架构在 SM2 加密运算中可实现每秒 209 次 256 位点乘计算， 资源占用为 59.7k LUT。该架构兼顾性能与能效，可满足物联网、区块链等场景下对 SM2 算法高速加密的需求。

每个模块的实现原理如下（从底层到高层）：

①乘法器：
128-bit 乘法器：booth编码乘法算法。
256-bit 乘法器：karatsuba分治乘法算法。

②模约减：
基于SM2算法参数的伪梅森素数性质专门设计的模约减单元，简单连接在256-bit乘法器之后构成模乘器，速度远快于传统模乘。

③点加和点倍运算：
本设计使用雅可比坐标系进行映射，经过对计算流程的调度，在双线并行计算的情况下，尽量减少运算周期。

④点乘运算：
通过多路选择器等进行判断来进行PA和PD模块的输入输出。

⑤模逆运算：
使用Radix-4二进制GCD模除Plus-Minus 算法（以扩展欧几里得算法为基础）。
